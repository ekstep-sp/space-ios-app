Java is one of the world's most popular programming languages (TIOBE Index). Running under the continuous process of enhancement and evolution, it has always taken into consideration the requirements of the past and the future.

The need for better adaptation of future releases and the needs of today's developers calls for improvements in the language, library, performance and learning curve. Taking this into consideration, the combined effort and collaboration of the industry, Oracle engineers and the Java community has led to the creation and release of Java 9.

Java 9 not only brings improvements to the language and the library, but also extensively remodels its whole system. This course will take you through the features and concepts of the new module system as well as the significant language and library updates.

 

Target Audience: Developers
